A dynamic Weather Dashboard that allows users to search any city and view both current weather conditions and a 5-day forecast. Built using the OpenWeather APIs and JavaScript, this app also keeps a searchable history of cities using `localStorage`.

## Features

- Search for any city to see:
  - Current weather: temperature, humidity, wind speed, and weather icon
  - 5-day forecast: date, weather icon, temperature, wind speed, and humidity
- Persistent search history with clickable buttons
- Responsive and interactive user interface
- Dynamic updates using server-side API calls

## Technologies Used

- HTML, CSS, JavaScript
- OpenWeather Geocoding API
- OpenWeather 5 Day / 3 Hour Forecast API
- `localStorage` for persistent search history

## Usage

1. Enter a city name in the search bar.
2. Click "Search" to fetch current and forecasted weather data.
3. Click a city from the search history to quickly view its weather again.
